# 📚 Study Planner — Term 3 Weighted Assessments

A lightweight study planner for my **Term 3 Weighted Assessments**. It creates a personalized day-by-day revision schedule based on subject band, Mother Tongue, and personal commitments.

✨ **No installation required** — just use github pages.

## 🚀 Features

* 📅 Generates a personalized study plan from the assessment schedule.
* 🎯 Filters subjects based on your **G1/G2/G3** band and Mother Tongue.
* 📝 Add homework, tuition, CCA, or other commitments.
* ⏰ Creates **45-minute study sessions** with **10-minute breaks**.
* 📈 Prioritizes subjects by assessment weight and upcoming test dates.
* 😌 Keeps Sundays as light review days.
* 💾 Saves everything locally using your browser's `localStorage`.
* 🌙 Automatically matches your device's light/dark appearance setting.

## 🛠️ Customizing

Assessment data is stored in the `TESTS` array at the top of `app.js`. Edit it to update schedules, subjects, or add future terms.
